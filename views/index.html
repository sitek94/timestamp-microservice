        {
          "unix": 1609286400000,
          "utc": "Wed, Dec 30, 2020, 01:00:00 GMT+1"
        }

Unix timestamp

        GET /api/timestamp/1451001600000
      
        {
          "unix": 1451001600000,
          "utc": "Fri, Dec 25, 2015, 01:00:00 GMT+1"
        }

Invalid date

        GET /api/hello-guys
      
        {
